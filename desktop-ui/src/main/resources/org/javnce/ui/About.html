Javnce - desktop sharing client and server.

The Javnce is for easily share desktop or connect to a shared desktop.

Sharing your desktop

Sharing your desktop with others is easy. You need only select following when asked:

  • Name

    Server name is shown to other Javnce users. The name is just for recognize the different servers from each others.

  • View access mode

    When you select this one then users can only see your desktop. Other users cannot not interact with your desktop.

  • Full access mode

    When you select this one then users can interact with your desktop, i.e. user have access to your mouse and keyboard and can do what ever you can do.

  • Auto-connect.

    If Auto-connect is not selected then you can either accept or decline users request to connect to your desktop.

    That is if you select Auto-connect then users can connect to your desktop without your interaction.

    Note that the accept or decline users request to connect is only "security" feature in Javnce.

Connecting to shared desktop

Connecting to shared desktop is easy.

The Javnce searches the shared desktop and you just select to with one you want to connect.
