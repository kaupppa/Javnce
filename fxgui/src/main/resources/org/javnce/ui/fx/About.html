Javnce - desktop sharing client and server.

The Javnce is for easy desktop sharing.

Sharing your desktop

When sharing your desktop you need to tell two things:

  • Name

    Name is used for recognize the different servers from each others.

  • Mode

    Defines if client can only see the desktop or if client interact your desktop using keyboard and mouse.

Connecting to shared desktop

Javnce searches the shared desktop. Just select to with one you want to connect.
